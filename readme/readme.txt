Plugin for CudaText, which shows current time (HH:MM:SS) in the statusbar,
in the new rightmost cell (new cell is added, just after the "message text"
wide cell).

Plugin has the config in the [CudaText]/settings/plugins.ini with the option:

[clock]
show_seconds=1

Possible values are 0 and 1 (boolean).
You can open the ini-file using the menu: "Options / Settings-plugins / Clock".
